# 1001. 两数求和 ★

## 题目描述

输入一行两个整数 `a` 和 `b`，输出它们的和。

示例：

```text
输入:
2 3

输出:
5
```

约束：

- `-100000 <= a, b <= 100000`

## 解题思路

这是一道最小可运行样例题。读取两个整数后直接相加并输出即可。

## Python 代码

```python
import sys


def main() -> None:
    a, b = map(int, sys.stdin.read().split())
    print(a + b)


if __name__ == "__main__":
    main()
```

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)
